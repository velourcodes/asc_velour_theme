# asc_velour_theme
ASC Velour - Affan Siddiqi Velour Theme! Doesnt just sound cool, but just like me, this theme also brings comfort, customization, premium experience and most vitally, unmatched comfort for a dev's eyes.

# Features
- Elegant dark theme with balanced contrast
- Optimized for long coding sessions
- Eye-comfort colors that reduce strain
- Works great with modern languages & frameworks

# Installation

## Method 1
- Open Extensions Marketplace in VS Code
- Search for ASC Velour Midnight Theme - By Affan
- Click Install
- Go to Preferences → Color Theme → ASC Velour Midnight Theme - By Affan

## Method 2
- Open VS Code on your machine
- Go to Settings → Themes → Color Theme → Install Additional Color Themes → Click On Three Dots on the Search (Left Side of your screen) → Choose the locally downloaded .vsix file
